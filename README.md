# istema-de-Gest-o-Escolar---API-REST
🎓 Sistema de Gestão Escolar API | Desenvolvida com Spring Boot 3 e Spring Security. O projeto demonstra a implementação completa de CRUDs para Alunos, Professores e Cursos, utilizando o padrão DTO (Data Transfer Object), validações de dados com Bean Validation e criptografia de dados sensíveis com BCrypt.
